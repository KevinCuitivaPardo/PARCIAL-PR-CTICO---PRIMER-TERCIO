## Compilar y ejecutar

### Compilar el proyecto
```
mvnw.cmd clean install
```

### Ejecutar la aplicación
```
mvnw.cmd spring-boot:run
```
### Ejecutar tests
```
 mvnw.cmd test
```

## Estructura del proyecto

